Sync your RPM data across all devices via a private GitHub Gist.

Setup (one time):
1. Go to github.com/settings/tokens/new
2. Set a name (e.g. "RPM Planner"), tick gist scope, click Generate
3. Copy the token and paste it below
4. Leave Gist ID blank on first push — it will be created automatically
